# Quotes Scraper

Web scraping application for extracting quotes and authors data from websites using Scrapy.

## Features

- Scraping quotes
- Scraping authors information
- Export data to JSON
- MongoDB integration
- Automatic data parsing
- Structured data storage

## Technologies

- Python
- Scrapy
- MongoDB
- MongoEngine
- JSON
